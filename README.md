# Customer Shopping Behavior Analysis Dashboard

## Project Overview

This project presents an end-to-end customer shopping behavior analysis using Python, SQL, and Power BI. The objective is to uncover customer demographics, purchasing patterns, loyalty trends, and key revenue drivers through data analysis and interactive visualizations.

## Tools & Technologies

- Python
- Pandas
- NumPy
- SQL (MySQL)
- Power BI
- DAX

## Project Workflow

1. Data Collection and Loading
2. Data Cleaning and Preprocessing using Python
3. Exploratory Data Analysis (EDA)
4. SQL-Based Business Queries
5. Interactive Dashboard Development in Power BI
6. Insight Generation and Business Recommendations

## Dashboard Features

- Customer Distribution by Gender
- Revenue by Category
- Revenue by Age Group
- Average Purchase by Age Group
- Customer Subscription Distribution
- Revenue by Payment Method
- Revenue by Customer Loyalty
- Interactive Filters and KPIs

## Key Insights

- Customers with 20+ previous purchases generate the highest revenue.
- Clothing is the highest revenue-generating category.
- Female customers slightly outnumber male customers.
- Subscription status influences customer purchasing behavior.
- Customer loyalty is strongly associated with revenue growth.

## Files Included

- `project.ipynb` – Data cleaning and exploratory analysis
- `customer_behavior.sql` – SQL queries and business analysis
- `customer_shopping_behavior.csv` – Dataset
- `customer behavior dashboard.pbix` – Power BI dashboard
- `customer behavior dashboard.pdf` – Dashboard export
